• Video Streaming Platform

    • Description: Developing a comprehensive backend for a video streaming platform.
    • Learnings:
      • User Authentication: Designed and implemented user registration, login, and logout functionalities
      • Database Management: Created and connected database schemas/models with MongoDB and Mongoose.
      • Middleware Development: Created authentication and Multer middleware for handling authentication and file uploads.
      • File Upload Handling: Utilized Multer for managing file uploads
      • API Development: Built and tested RESTful APIs using Postman collections.
      • Error Handling: Developed custom API error handling and response utilities.
      • Aggregation Pipelines: Implemented aggregation pipelines and subpipelines for efficient data retrieval in watch history and channel profiles.
    • Technologies Used: Node.js, Express.js, MongoDB, Mongoose, JWT, bcrypt, Multer, Postman, Git, Cloudinary.
